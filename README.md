# TMF931 Open Gateway Onboarding And Ordering Component Suite

TM Forum Open API (Apache 2.0) Open Gateway Onboarding and Ordering Component Suite.

This Operate API covers the onboarding operations between Channel Partners and Operators as described by GSMA. It offers a consolidation of multiple TM Forum based APIs into a single specification (a component suite). This API focusses on:

- Application Owner (Developer) Management (acceptance, onboarding, lifecycle management)

- Application Management (acceptance, onboarding, lifecycle management)

- Service API Management (ordering, retrieving)

The resources in this repository are a mirror of the official versions, published here for your convenience.
The authoritative source of information for this API is: https://www.tmforum.org/oda/open-apis/directory/
